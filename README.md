
# ClearSpeak AI
**AI-Based Public Speaking Feedback Using Speech Pattern Analysis**

## Project Overview
This project focuses on improving public speaking feedback using AI to analyze short recordings for patterns like filler word usage and speaking rhythm.

## Repository Structure
* `/data`: Audio samples and labeled features.
* `/src`: Python scripts for extraction and analysis.
* `/docs`: LaTeX source files and PDF report.
* `/presentation`: Slides and project visuals.
